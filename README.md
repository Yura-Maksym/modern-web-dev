# Лабораторна робота №1: Шаблони проєктування
**Курс:** Modern Web Development Technologies

Це виконання першої лабораторної роботи, де я реалізував три патерни на мові Python.

## 1. Factory Method (Фабричний метод)
- **Проблема:** Коли ми не знаємо наперед, які саме об'єкти нам знадобляться (наприклад, який транспорт вибере клієнт).
- **Ідея:** Створити спеціальний метод, який сам вирішує, який об'єкт створити, щоб не прописувати це вручну кожного разу.
- **Де застосувати:** У системах замовлень, логістиці або при роботі з різними типами документів.

## 2. Facade (Фасад)
- **Проблема:** Коли бібліотека або система занадто складна, і щоб зробити одну дію, треба викликати 10 різних функцій.
- **Ідея:** Зробити одну "головну кнопку" (клас-фасад), яка сама запустить всі потрібні процеси всередині.
- **Де застосувати:** Обробка відео, робота зі складними базами даних.

## 3. Observer (Спостерігач)
- **Проблема:** Треба автоматично розсилати сповіщення всім користувачам, коли щось змінюється (наприклад, з'явилась нова новина).
- **Ідея:** Створити список підписників. Як тільки щось трапляється, програма сама "пробігає" по списку і сповіщає кожного.
- **Де застосувати:** Соціальні мережі, новинні портали, чати.
